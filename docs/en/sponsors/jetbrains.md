## Who we are

JetBrains was founded in **February 2000** (originally as *IntelliJ Software*) by three engineers — **Sergey Dmitriev**, **Valentin Kipyatkov**, and **Eugene Belyaev** — and is headquartered in **Prague (Czech Republic)** with global offices (including Amsterdam, Munich, Boston, and Saint Petersburg). It is a **bootstrapped** company — no external capital — with around **2,800 employees** serving more than **11 million active users** and **287,000+ business customers** worldwide.

The company specializes in **intelligent tools for software developers and teams** — IDEs and languages that cut repetitive work, suggest safe refactorings, automate testing, and give instant feedback on code quality.

## Flagship products

**IDEs**

- **IntelliJ IDEA** — JVM (Java, Kotlin, Scala, Groovy)
- **GoLand** — Go
- **PyCharm** — Python
- **WebStorm** — JavaScript / TypeScript
- **Rider** — .NET / C# / Unity
- **CLion** — C / C++
- **PhpStorm** — PHP
- **RubyMine** — Ruby / Rails
- **DataGrip** — databases
- **Fleet** — next-generation polyglot editor
- **ReSharper**, **dotTrace**, **dotMemory** — .NET productivity

**Language**

- **Kotlin** — created by JetBrains in **2011** (open-sourced in 2012); the **official language for Android development since 2019**.

**Teams & collaboration**

- **YouTrack** (issue tracking), **Space** (collaboration), **TeamCity** (CI/CD), **Qodana** (static analysis), and the **JetBrains AI Assistant**.

## Why it matters

Over two decades JetBrains has built its reputation by deepening what an IDE can understand about your code — cross-cutting type inference, deep code analysis, semantic navigation, and refactorings that respect language invariants. That "tools that know what they're talking about" philosophy makes JetBrains a natural partner for any challenge that measures efficiency, performance, and engineering discipline.
