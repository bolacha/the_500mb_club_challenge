## Quem somos

A JetBrains foi fundada em **fevereiro de 2000** (originalmente como *IntelliJ Software*) por três engenheiros — **Sergey Dmitriev**, **Valentin Kipyatkov** e **Eugene Belyaev** — com sede em **Praga (República Tcheca)** e escritórios globais (incluindo Amsterdã, Munique, Boston e São Petersburgo). É uma empresa **bootstrapped**, sem capital externo, com cerca de **2.800 colaboradores** atendendo mais de **11 milhões de usuários ativos** e **287.000+ empresas** clientes em todo o mundo.

Sua especialidade é criar **ferramentas inteligentes para desenvolvedores e times** — IDEs e linguagens que reduzem trabalho repetitivo, sugerem refatorações seguras, automatizam testes e dão *feedback* instantâneo sobre qualidade de código.

## Principais produtos

**IDEs**

- **IntelliJ IDEA** — JVM (Java, Kotlin, Scala, Groovy)
- **GoLand** — Go
- **PyCharm** — Python
- **WebStorm** — JavaScript / TypeScript
- **Rider** — .NET / C# / Unity
- **CLion** — C / C++
- **PhpStorm** — PHP
- **RubyMine** — Ruby / Rails
- **DataGrip** — bancos de dados
- **Fleet** — editor multilíngue de nova geração
- **ReSharper**, **dotTrace**, **dotMemory** — produtividade em .NET

**Linguagem**

- **Kotlin** — criada pela JetBrains em **2011** (open-source desde 2012); linguagem **oficial para desenvolvimento Android desde 2019**.

**Times & colaboração**

- **YouTrack** (issue tracking), **Space** (colaboração), **TeamCity** (CI/CD), **Qodana** (análise estática) e o assistente de IA **JetBrains AI Assistant**.

## Por que importa

A JetBrains construiu duas décadas de reputação aprofundando o que uma IDE pode entender sobre seu código — inferência de tipos cruzada, *deep code analysis*, navegação semântica e refatorações que respeitam invariantes da linguagem. É essa filosofia "ferramentas que sabem do que falam" que torna a JetBrains uma parceira natural para qualquer challenge que mede eficiência, performance e disciplina de engenharia.
